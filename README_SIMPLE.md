# The Conch

Chat with an intelligent conch shell that preserves cultural heritage in Amphitopia, an underwater colony in 2080.

## Run It (One Command)

```bash
./start.sh
```

That's it! The first time will take 2-3 minutes to install. After that, it starts instantly.

## What You Get

- AI-powered responses from a cultural preservation device
- The Conch explains land concepts to underwater colony residents
- Beautiful terminal interface with atmospheric effects
- Stays 100% in character as a conch artifact

## Example

```
You: What is the sky?

The sky is a protective layer that is very far away, and may
change colors depending on the universe's mood. Think of it
as the dome above Amphitopia, but natural, infinite, and
constantly shifting.
```

## Exit

Type `exit` or press `Ctrl+C`
